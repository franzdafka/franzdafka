### Projects

**[Credit Risk Assessment Service](https://github.com/franzdafka/credit-risk-analysis)**
Credit scoring API built with FastAPI and scikit-learn. Includes SHAP-based decision explanations, model benchmarking (Logistic Regression, Random Forest, GBM, XGBoost), Docker deployment, and a full EDA notebook with industry-standard metrics (Gini, KS statistic, lift curve).

**[Portfolio Optimization — Markowitz Mean-Variance](https://github.com/franzdafka/portfolio-optimization)**
Mean-variance optimization from scratch in Python: efficient frontier, Global Minimum Variance and Maximum Sharpe portfolios via constrained quadratic programming. Applied to real S&P 500 data with Monte Carlo simulation (8,000 portfolios) and sensitivity analysis on estimation error.
